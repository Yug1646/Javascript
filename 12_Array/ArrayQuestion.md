# JavaScript Arrays – Practice Questions

These questions are **based only on what you have learned so far**.
Try to solve them **without looking back at notes**. Difficulty increases gradually.

---

## 🟢 Level 1: Basics (Warm-up)

1. Create an array of 5 fruits and print the first and last element.
2. Add an element at the end of an array using `push()`.
3. Remove the first element from an array and print the removed value.
4. Check whether a given variable is an array.
5. Copy an array without modifying the original one.

---

## 🟢 Level 2: Accessing & Destructuring

6. Use array destructuring to extract the first two elements of an array.
7. Use destructuring to skip the second element of an array.
8. Swap two variables using array destructuring.
9. Access an element from a nested array.
10. Use the rest operator to store remaining elements of an array.

---

## 🟡 Level 3: Array Methods (Core)

11. Use `concat()` to merge two arrays.
12. Convert an array of names into a single string using `join()`.
13. Check if a specific value exists in an array using `includes()`.
14. Find the index of an element using `indexOf()`.
15. Reverse an array and observe whether the original array changes.

---

## 🟡 Level 4: Transformation & Search

Use the `customers` array for the following questions.

16. Get all customers whose age is greater than 30.
17. Create a new array that contains only the expenses of customers.
18. Add a `full_name` field to each customer.
19. Find the first customer whose age is less than 30.
20. Find the index of the last customer whose age is less than 30.

---

## 🟠 Level 5: Reduce & Logic

21. Calculate the sum of numbers in an array using `reduce()`.
22. Calculate the average age of customers who purchased a "Book".
23. Calculate the total expense of married customers.
24. Count how many customers are unmarried.

---

## 🟠 Level 6: Boolean Checks

25. Check if **any** customer is under 25 years old.
26. Check if **all** customers are married.
27. Check if at least one customer spent more than 1500.

---

## 🔵 Level 7: Iterators & Loops

28. Use `entries()` with `for...of` to print index and value of an array.
29. Use `values()` to loop through an array.
30. Explain what happens when you call `.next()` on an iterator before looping.

---

## 🔵 Level 8: Advanced Array Methods

31. Flatten a nested array using `flat()`.
32. Use `flatMap()` to double each number in an array.
33. Group customers by `dept` using `Object.groupBy()`.
34. Create a new array using `toSorted()` without modifying the original array.
35. Replace an element in an array using `with()`.

---

## 🟣 Level 9: Chaining (Most Important)

36. Get the total expense of unmarried customers using **method chaining**.
37. Get the names of customers older than 30 using chaining.
38. Calculate the average expense of customers younger than 35.

---

## ⭐ Bonus (Think Before Coding)

39. Difference between `map()` and `forEach()`?
40. Difference between `slice()` and `splice()`?
41. Why is `reduce()` considered powerful?
